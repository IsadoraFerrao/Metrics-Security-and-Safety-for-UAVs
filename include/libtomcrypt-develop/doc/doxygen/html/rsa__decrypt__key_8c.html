Parameters
inThe ciphertext
inlenThe length of the ciphertext (octets)
out[out] The plaintext
outlen[in/out] The max size and resulting size of the plaintext (octets)
lparamThe system "lparam" value
lparamlenThe length of the lparam value (octets)
hash_idxThe index of the hash desired
paddingType of padding (LTC_PKCS_1_OAEP or LTC_PKCS_1_V1_5)
stat[out] Result of the decryption, 1==valid, 0==invalid
keyThe corresponding private RSA key
Returns
CRYPT_OK if succcessul (even if invalid)
