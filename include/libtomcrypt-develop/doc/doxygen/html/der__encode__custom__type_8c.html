Encode a Custom Type.

This function is a bit special compared to the others, as it requires the root-ltc_asn1_list where the type is defined.

Parameters
rootThe root of the list of items to encode
out[out] The destination
outlen[in/out] The size of the output
Returns
CRYPT_OK on success
