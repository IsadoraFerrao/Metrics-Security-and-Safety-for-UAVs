Convert data from a specific radix to binary.

The default MPI descriptors ltm_desc, tfm_desc and gmp_desc have the following restrictions on parameters:

in - NUL-terminated char buffer

radix - 2..64

Parameters
inThe input
radixThe radix of the input
outThe output buffer
len[in/out] The length of the output buffer
Returns
CRYPT_OK on success.
