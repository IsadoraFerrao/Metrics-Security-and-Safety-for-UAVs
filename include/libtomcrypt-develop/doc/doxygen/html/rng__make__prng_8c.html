Create a PRNG from a RNG.

In case you pass bits as '-1' the PRNG will be setup as if the export/import functionality has been used, but the imported data comes directly from the RNG.

Parameters
bitsNumber of bits of entropy desired (-1 or 64 ... 1024)
wprngIndex of which PRNG to setup
prng[out] PRNG state to initialize
callbackA pointer to a void function for when the RNG is slow, this can be NULL
Returns
CRYPT_OK if successful
